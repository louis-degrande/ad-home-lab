# Active Directory lab setup

# Goal
I'm building this lab to get hands-on expertience with the core concepts behind enterprise networks.
Active Directory is used in (almost) every business network and understanding how it works is also the foundation for cybersecurity skills like detecting lateral movement and privilege escalation.

# Tools used
- Oracle Virtualbox
- Windows Server 2022 (evaluation)
- Windows 10 (evaluation)
- draw.io (network diagram)
